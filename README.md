# JamRoom Drum Machine v5

Focused MPC drum machine with live looping.

## Added in v5

- Count-In: Off, 1 Bar, 2 Bars, 4 Bars
- Big visual count-in overlay
- High downbeat click and lower beat clicks
- Recording starts automatically after count-in
- Existing loop can keep playing while count-in runs
- Grid menu simplified to 1/4 Beat, 1/8 Beat, 1/16 Beat, 1/32 Beat
- Snap applies to mouse-added sequencer notes and recorded pad hits

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
